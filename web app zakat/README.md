# Web App Zakat - Digital Alms Management System

Web App Zakat adalah aplikasi berbasis web yang dirancang untuk mendigitalisasi pengelolaan zakat, infaq, dan sedekah. Sistem ini bertujuan untuk mempermudah perhitungan, pengumpulan, serta penyaluran dana sosial keagamaan secara transparan dan akuntabel.

## **Visi Proyek**

Meningkatkan efisiensi dan kepercayaan publik dalam pengelolaan zakat dengan menyediakan platform yang memiliki standar perhitungan akurat sesuai kaidah, serta memastikan setiap distribusi dana dapat terdokumentasi dengan baik untuk dilaporkan kepada para muzakki (pemberi zakat).

## **Fitur Utama**

- **Muzakki & Mustahik Management:** Database terstruktur untuk mengelola data pemberi zakat dan penerima manfaat guna memastikan distribusi yang tepat sasaran.
- **Real-time Distribution Reporting:** Dashboard laporan yang menyajikan status penyaluran dana secara transparan bagi pengelola organisasi.
- **Transaction History:** Pencatatan riwayat transaksi yang rapi untuk mempermudah proses audit tahunan.

## **Teknologi yang Digunakan**

- **Core Framework:** SvelteKit
- **Database:** Serverlss PostgreSQL
