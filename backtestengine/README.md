# Backtest Engine - Quantitative Strategy Validator

Backtest Engine adalah mesin simulasi yang memungkinkan trader dan analis untuk menguji hipotesis dan thesis ideas dari market menggunakan data historis yang luas sebelum masuk ke pasar riil.

## **Visi Proyek**

Memberikan validasi empiris berbasis data terhadap sebuah strategi trading melalui pengolahan data besar, sehingga pengguna dapat memahami ekspektasi profitabilitas (_expectancy_) dan karakteristik risiko dari strategi tersebut secara objektif.

## **Fitur Utama**

- **Multi-Instrument Testing:** Mendukung pengujian pada berbagai aset CFD.
- **Performance Metrics Analytics:** Kalkulasi otomatis untuk Win Rate, Maximum Drawdown, Sharpe Ratio, dan Profit Factor.
- **Equity Curve Visualization:** Representasi visual pertumbuhan modal untuk mengidentifikasi fase stagnasi atau lonjakan risiko.
- **Machine Learning Based Pattern Finder:** Menggunakan machine learning untuk menemukan pola yang sering muncul pada market yang akan digunakan untuk strategi trading.

## **Teknologi yang Digunakan**

- **Data Processing:** Python (Pandas, NumPy)
- **Visualization:** Matplotlib / Plotly
